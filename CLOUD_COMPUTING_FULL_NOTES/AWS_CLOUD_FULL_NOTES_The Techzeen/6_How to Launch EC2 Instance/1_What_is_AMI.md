# What is AMI?

AMI stands for **Amazon Machine Image**.

In simple words:

AMI is a ready-made template of a computer that contains: - Operating
System (like Linux or Windows) - Required software - Basic configuration
settings

When we create an EC2 instance, we must select an AMI first.

Think of it like this:

-   AMI = Blueprint of the computer\
-   EC2 = Actual computer created from that blueprint

Without AMI, we cannot launch an EC2 instance.
