
✅ Step 1: Launch an EC2 Instance
Login to AWS Console.
Go to Amazon EC2.
Click Launch Instance.
Choose Amazon Linux (or Ubuntu).
Select instance type (e.g., t2.micro – free tier).
Allow HTTP (80) and SSH (22) in Security Group.
Launch and download the key pair.







✅ Step 2: Connect to EC2 Server
From your local machine:
ssh -i your-key.pem ec2-user@your-public-ip

(For Ubuntu use ubuntu@your-public-ip)



✅ Step 3: Update the Server

sudo yum update -y   # For Amazon Linux

For Ubuntu:
sudo apt update -y






✅ Step 4: Install Node.js
For Amazon Linux:
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo yum install -y nodejs

Verify:
node -v
npm -v





✅ Step 5: Upload Your Node.js App

You can:
Clone from GitHub:
git clone https://github.com/your-repo.git
cd your-repo

OR
Upload using SCP:
scp -i your-key.pem -r myapp ec2-user@your-public-ip:/home/ec2-user/





✅ Step 6: Install App Dependencies
Inside your project folder:
npm install
Start the app:
node app.js

Test:
http://your-public-ip:3000

(Assuming app runs on port 3000)







✅ Step 7: Install PM2 (Keep App Running)
Install PM2 globally:
sudo npm install -g pm2
Start app with PM2:
pm2 start app.js
pm2 save
pm2 startup


PM2 ensures your app runs even if the server restarts.






✅ Step 8: Install NGINX
Install NGINX:
Amazon Linux:
sudo yum install nginx -y

Ubuntu:
sudo apt install nginx -y

Start NGINX:
sudo systemctl start nginx
sudo systemctl enable nginx


Check in browser:
http://your-public-ip

You should see the NGINX welcome page.





✅ Step 9: Configure NGINX as Reverse Proxy
Edit NGINX config:
sudo nano /etc/nginx/nginx.conf

Or create a new config file:
sudo nano /etc/nginx/conf.d/nodeapp.conf

Add:
server {
    listen 80;
    server_name your_public_ip;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}

Save and restart:
sudo systemctl restart nginx






✅ Step 10: Test Final Deployment

Now open:
http://your-public-ip

🎉 Your Node.js app is now running through NGINX on port 80!
