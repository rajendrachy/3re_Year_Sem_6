# Practical 2: Create EC2 Instance (Windows) and Connect Using RDP

## Steps:

1.  Launch EC2 Instance
2.  Select AMI: Windows Server
3.  Choose Instance Type: t2.micro / t3.micro
4.  Create Key Pair
5.  Enable Public IP
6.  Add Security Group Rule:
    -   RDP (3389) from Anywhere
7.  Launch Instance

## Connect Using RDP:

1.  Go to EC2 → Select Instance → Connect → RDP Client
2.  Download RDP file
3.  Get Windows Password (using key pair)
4.  Open RDP file and login using Administrator credentials
