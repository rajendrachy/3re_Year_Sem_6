# 1. Hypervisor

## Step 1: Launch a Linux Ubuntu Virtual Machine using VMware Workstation Player

- Download and install VMware Workstation Player  
- Download the Ubuntu ISO file  
- Open VMware Player and click on “Create a New Virtual Machine”  
- Select Installer disc image file (ISO) and choose Ubuntu ISO  
- Select Linux → Ubuntu (64-bit)  
- Enter virtual machine name and location  
- Allocate disk size (minimum 20 GB)  
- Customize RAM and processor if needed  
- Click Finish and then click “Play Virtual Machine”  
- Complete Ubuntu installation  

---

## Step 2: Connect to Virtual Machine

- Open VMware Player or Hyper-V Manager  
- Select the Ubuntu virtual machine  
- Click on Start / Power On  
- Wait for system to boot  
- Enter username and password  
- Virtual machine is now connected  

---

## Step 3: Create a directory by your name

- Open Terminal in Ubuntu VM  
- Type command: `mkdir YourName`  
- Press Enter  
- Directory will be created in current location  

---

## Step 4: Change RAM configuration (Add 2GB)

- Shut down the virtual machine  
- Open VMware Player and select VM  
- Click on “Edit Virtual Machine Settings”  
- Select Memory option  
- Increase RAM by 2GB (example: 2GB → 4GB)  
- Click OK to save changes  
- Restart the virtual machine  

---

**Result:** Ubuntu VM is successfully created, connected, directory is created, and RAM is updated.
