# ☁️ Complete Roadmap to Learn Cloud Computing

This roadmap provides a structured path to become a Cloud Engineer / Cloud Architect.

---

#  Phase 1: Basics (Foundation)

## 1️ Computer Fundamentals
- Operating Systems (Windows/Linux)
- Basic Networking
- Storage concepts
- Virtualization basics

## 2️ Networking Fundamentals
- IP Address
- DNS
- DHCP
- HTTP/HTTPS
- TCP/IP
- Firewalls
- Load Balancer

 Networking is very important in cloud.

---

#  Phase 2: Linux & Scripting

## 3️ Linux
- File system
- Permissions
- Users & groups
- Package management
- Basic commands
- SSH

## 4️ Scripting
- Bash scripting
- Basic Python (recommended)
- Automation basics

---

#  Phase 3: Cloud Fundamentals

## 5️ Cloud Computing Concepts
- What is Cloud Computing?
- IaaS, PaaS, SaaS
- Public, Private, Hybrid Cloud
- On-Premises vs Cloud
- Regions & Availability Zones
- Scalability & Elasticity
- CapEx vs OpEx

---

## 6️ Version Control
