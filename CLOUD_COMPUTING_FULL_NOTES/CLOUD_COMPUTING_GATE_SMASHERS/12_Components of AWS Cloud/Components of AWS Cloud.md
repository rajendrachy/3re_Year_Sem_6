# 1️⃣ AWS Compute Services (00:45)

Compute services provide processing power to run applications in the cloud.

## 🔹 Amazon EC2 (Elastic Compute Cloud)
- Virtual servers in the cloud  
- Full control over operating system  
- Scalable capacity  

## 🔹 AWS Lambda
- Serverless compute service  
- Runs code without managing servers  
- Pay only for execution time  

## 🔹 Amazon ECS (Elastic Container Service)
- Container management service  
- Runs Docker containers  

## 🔹 Amazon EKS (Elastic Kubernetes Service)
- Managed Kubernetes service  
- Used for container orchestration  

## 🔹 AWS Elastic Beanstalk
- Platform for deploying applications easily  
- Automatically handles scaling  

👉 Compute services are used to host websites, applications, APIs, and backend systems.

---

# 2️⃣ AWS Storage Services (05:00)

Storage services allow users to store data securely and reliably.

## 🔹 Amazon S3 (Simple Storage Service)
- Object storage service  
- Highly durable and scalable  
- Used for backups, static websites, media storage  

## 🔹 Amazon EBS (Elastic Block Store)
- Block storage for EC2 instances  
- Used as virtual hard drive  

## 🔹 Amazon EFS (Elastic File System)
- Managed file storage  
- Shared storage for multiple instances  

## 🔹 Amazon Glacier
- Low-cost archival storage  
- Used for long-term backups  

👉 AWS offers different storage types based on speed, cost, and use case.

---

# 3️⃣ AWS Database Services (07:02)

AWS provides managed database services.

## 🔹 Amazon RDS (Relational Database Service)
- Managed SQL databases  
- Supports MySQL, PostgreSQL, SQL Server, etc.  

## 🔹 Amazon DynamoDB
- NoSQL database  
- Fully managed and highly scalable  

## 🔹 Amazon Aurora
- High-performance relational database  
- Compatible with MySQL and PostgreSQL  

## 🔹 Amazon Redshift
- Data warehouse service  
- Used for analytics and big data processing  

👉 Database services reduce maintenance effort and improve scalability.

---

# 4️⃣ Networking & Content Delivery (08:32)

Networking services connect and secure cloud resources.

## 🔹 Amazon VPC (Virtual Private Cloud)
- Create isolated cloud networks  
- Control IP ranges, subnets, routing  

## 🔹 Elastic Load Balancer (ELB)
- Distributes traffic across servers  
- Improves availability  

## 🔹 Amazon CloudFront
- Content Delivery Network (CDN)  
- Delivers content globally with low latency  

## 🔹 Route 53
- DNS (Domain Name System) service  
- Routes users to applications  

👉 Networking ensures performance, availability, and secure communication.

---

# 5️⃣ Security, Monitoring & AI Tools (10:04)

## 🔐 Security Services

### 🔹 AWS IAM (Identity and Access Management)
- Manages users and permissions  
- Role-based access control  

### 🔹 AWS Shield
- Protection against DDoS attacks  

### 🔹 AWS WAF (Web Application Firewall)
- Protects web applications  

---

## 📊 Monitoring Services

### 🔹 Amazon CloudWatch
- Monitors resources and applications  
- Provides logs and metrics  

### 🔹 AWS CloudTrail
- Tracks API activity  
- Helps in auditing and compliance  

---

## 🤖 AI & Machine Learning Tools

### 🔹 Amazon SageMaker
- Build and deploy ML models  

### 🔹 AWS Rekognition
- Image and video analysis  

### 🔹 Amazon Polly
- Text-to-speech service  

👉 AWS integrates AI tools directly into cloud infrastructure.
