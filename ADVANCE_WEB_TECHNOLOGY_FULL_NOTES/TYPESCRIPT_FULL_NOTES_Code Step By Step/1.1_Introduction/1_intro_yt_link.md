##  What is TypeScript?
TypeScript is a **statically typed superset of JavaScript** that compiles into plain JavaScript. :contentReference[oaicite:0]{index=0}  

- Adds **type safety**
- Helps catch errors **before runtime**
- Improves **code readability & maintainability**
- Works with all JavaScript frameworks

---

##  Why Learn TypeScript?
- Detect errors early (compile-time checking)
- Better **auto-completion & IDE support**
- Scalable for large applications
- Cleaner and more structured code

---

##  Key Features
- Static Typing
- Type Inference
- Interfaces & Types
- Generics
- Enums
- Modern JavaScript support
