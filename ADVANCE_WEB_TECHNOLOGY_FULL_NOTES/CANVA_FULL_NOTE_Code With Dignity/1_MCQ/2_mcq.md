# HTML5 Canvas – 30 MCQs

## Instructions

* Each question has four options.
* The correct answer is marked.
* A brief explanation is provided.

---

### 1. What does the <canvas> element in HTML5 provide?

A. Database storage
B. A drawing surface for graphics
C. Audio playback
D. Form validation

**Answer:** B
**Reason:** The canvas element is used to draw graphics via JavaScript.

---

### 2. Which attribute sets the width of a canvas?

A. size
B. width
C. style-width
D. dimension

**Answer:** B
**Reason:** The width attribute directly defines the canvas width.

---

### 3. What is the default canvas size?

A. 500x500
B. 800x600
C. 300x150
D. 1024x768

**Answer:** C
**Reason:** By default, canvas width is 300px and height is 150px.

---

### 4. Which method is used to get the drawing context?

A. getContext()
B. getCanvas()
C. context()
D. createContext()

**Answer:** A
**Reason:** getContext() returns the drawing context like 2D.

### 5. Which context type is most commonly used?

**Reason:** The 2D context is most widely used for drawing shapes.

### 6. How do you draw a rectangle?
