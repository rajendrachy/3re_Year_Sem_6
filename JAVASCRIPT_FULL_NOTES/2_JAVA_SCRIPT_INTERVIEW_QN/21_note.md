| Feature              | Client-Side Rendering (CSR)                     | Server-Side Rendering (SSR)                     |
|---------------------|------------------------------------------------|------------------------------------------------|
| Rendering Location  | Browser (Client)                               | Server                                         |
| Initial Load Time   | Slow (loads JS first)                          | Fast (HTML already rendered)                   |
| SEO                 | Poor (content loads later)                     | Good (content available immediately)           |
| Performance         | Fast after initial load                        | Slower navigation (page reloads)               |
| Server Load         | Low                                            | High                                          |
| JavaScript Required | Yes                                            | Not strictly required                          |
| User Experience     | Smooth (SPA-like)                              | Traditional page reload                        |
| Examples            | React, Angular                                 | Next.js, Django                                |
