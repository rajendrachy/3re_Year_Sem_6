## 28. What is DOM (Document Object Model)?

The **DOM (Document Object Model)** is a programming interface that represents a web page in a **tree-like structure** where every element, attribute, and piece of text is an **object** that can be accessed and manipulated using JavaScript. Think of it as a **bridge between HTML and JavaScript** — it allows scripts to **read, modify, add, or delete content, structure, and styles** on a webpage dynamically.

### Key Features of DOM:
1. **Tree Structure**
   - The entire HTML document is represented as a **hierarchical tree**.
   - The **document object** is the root, and every element (like `<div>`, `<p>`, `<h1>`) becomes a **node** in this tree.
   - Example structure:
     ```
     document
       ├── html
           ├── head
           └── body
               ├── h1
               └── p
     ```

2. **Dynamic Interaction**
   - With DOM, you can **update content, styles, and structure** without refreshing the page.
   - Example actions:
     - Change text or HTML content.
     - Add or remove elements.
     - Update CSS styles dynamically.

3. **Accessing Elements**
   - JavaScript provides several ways to **find elements in the DOM**:
     - `getElementById()` – Select an element by its ID.
     - `getElementsByClassName()` – Select elements by class name.
     - `querySelector()` / `querySelectorAll()` – Use CSS selectors.
     - `getElementsByTagName()` – Select elements by tag name.

4. **Manipulating Elements**
   - DOM allows you to **create, delete, or modify nodes**.
   - Example:
     ```javascript
     const para = document.createElement('p'); // Create a new paragraph
     para.innerText = 'Hello, DOM!';
     document.body.appendChild(para);         // Add it to the body
     ```

5. **Event Handling**
   - DOM provides **event-driven interaction**, like responding to clicks, keyboard input, or mouse movements.
   - Example:
     ```javascript
     document.getElementById('btn').addEventListener('click', function() {
       alert('Button clicked!');
     });
     ```

6. **Cross-Browser Standard**
   - DOM is standardized by the **W3C**, so most modern browsers support it in a similar way.
   - Provides a consistent way to **interact with web pages** programmatically.

### Why DOM is Important:
- Makes web pages **interactive** and **dynamic**.
- Allows **real-time updates** without refreshing the page.
- Forms the foundation for **modern front-end frameworks** like React, Angular, and Vue, which rely heavily on manipulating the DOM efficiently.

```javascript
// HTML: <p id="demo">Old Text</p>
document.getElementById('demo').innerText = 'New Text';
// The paragraph content now changes to "New Text" dynamically
