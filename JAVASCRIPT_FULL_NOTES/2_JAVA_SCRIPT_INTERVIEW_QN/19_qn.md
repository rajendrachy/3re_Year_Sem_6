# 1. What are Arrow Functions in JavaScript?

**Arrow functions** are a concise way to write functions in JavaScript, introduced in **ES6 (ECMAScript 2015)**. They provide a shorter syntax compared to traditional function expressions and **lexically bind the `this` keyword**, making them particularly useful in callbacks and object methods.

---

## Key Features

1. **Concise Syntax**
   - No `function` keyword.
   - Parentheses can be omitted for **single parameters**.
   - Braces and `return` can be omitted for **single-expression functions**.

2. **Lexical `this`**
   - Arrow functions do **not have their own `this`**.
   - They inherit `this` from the surrounding scope, preventing common issues in callbacks.

3. **Cannot Be Used as Constructors**
   - Arrow functions **cannot be called with `new`**.
   - They do not have a `prototype`.

4. **No `arguments` Object**
   - Arrow functions do not have their own `arguments`.
   - Use **rest parameters** (`...args`) instead.

---

## Syntax Examples

**Basic Example:**
```javascript
// Traditional function
function add(a, b) {
  return a + b;
}

// Arrow function
const add = (a, b) => a + b;
