# JavaScript Interview Question

## 15. Explain `call()`, `apply()`, and `bind()` methods

In JavaScript, `call()`, `apply()`, and `bind()` are **methods used to set the `this` value explicitly** for a function.

---

## 1. call()

- **Syntax:** `func.call(thisArg, arg1, arg2, ...)`  
- Calls a function **immediately** with a specified `this` value and individual arguments.

### Example

```javascript
function greet(greeting, punctuation) {
  console.log(greeting + ", " + this.name + punctuation);
}

let person = { name: "Rahul" };

greet.call(person, "Hello", "!");
```

Output:

```
Hello, Rahul!
```

---

## 2. apply()

- **Syntax:** `func.apply(thisArg, [argsArray])`  
- Calls a function **immediately** with a specified `this` value and **arguments as an array**.

### Example

```javascript
function greet(greeting, punctuation) {
  console.log(greeting + ", " + this.name + punctuation);
}

let person = { name: "Amit" };

greet.apply(person, ["Hi", "?"]);
```

Output:

```
Hi, Amit?
```

**Difference between `call` and `apply`:**  
- `call()` → arguments passed individually  
- `apply()` → arguments passed as an array

---

## 3. bind()

- **Syntax:** `func.bind(thisArg, arg1, arg2, ...)`  
- Returns a **new function** with `this` bound to the specified object.  
- Does **not call the function immediately**.

### Example

```javascript
function greet(greeting) {
  console.log(greeting + ", " + this.name);
}

let person = { name: "Anjali" };

let greetPerson = greet.bind(person);
greetPerson("Hello");
```

Output:

```
Hello, Anjali
```

- `bind()` is useful when you want to **pass a function around with a fixed `this` value**.

---

## Summary Table

| Method | Calls Function Immediately? | Arguments | Returns |
|--------|----------------------------|-----------|---------|
| call() | Yes | Comma-separated | Result of function |
| apply() | Yes | Array | Result of function |
| bind() | No | Comma-separated | New function |

---

## Short Interview Answer

- `call()` → Calls a function immediately with a specified `this` and arguments individually.  
- `apply()` → Calls a function immediately with a specified `this` and arguments as an array.  
- `bind()` → Returns a new function with `this` bound, does not call immediately.
