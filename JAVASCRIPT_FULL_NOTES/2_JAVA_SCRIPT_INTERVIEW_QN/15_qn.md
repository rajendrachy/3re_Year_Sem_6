# JavaScript Interview Question

## 17. What is Currying in JavaScript?

### Definition

**Currying** is a technique in JavaScript where a **function with multiple arguments is transformed into a sequence of functions**, each taking **one argument at a time**.

- Instead of passing all arguments at once, each function **returns another function** that takes the next argument.
- Helps in **reusing functions** and creating **specialized functions** from a general one.

---

### Example 1: Basic Currying

```javascript
function sum(a) {
  return function(b) {
    return a + b;
  }
}

let add5 = sum(5);    // returns a function that adds 5
console.log(add5(3)); // Output: 8
```

Explanation:
- `sum(5)` returns a new function that **adds 5** to its argument
- `add5(3)` → 5 + 3 = 8

---

### Example 2: Currying with Arrow Functions

```javascript
const multiply = a => b => a * b;

let double = multiply(2);
console.log(double(5)); // Output: 10
```

- `multiply(2)` returns a function that multiplies any number by 2

---

### Advantages of Currying

1. **Reusability** – Create specialized functions from general ones
2. **Function composition** – Easy to combine functions
3. **Partial application** – Fix some arguments and reuse the function

---

### Short Interview Answer

**Currying** in JavaScript is a technique where a function with multiple arguments is **transformed into a sequence of functions**, each taking **one argument at a time**, allowing **partial application and better function reuse**.
