# JavaScript Interview Question

## 10. What is an Immediately Invoked Function in JavaScript?

### Definition

An **Immediately Invoked Function Expression (IIFE)** is a function that **runs immediately after it is defined**.

It is used when we want to **execute a function once without calling it again**.

---

## Syntax

```javascript
(function () {
  // code here
})();
```

The function is wrapped in **parentheses** and then **called immediately** using `()`.

---

## Example

```javascript
(function () {
  console.log("This function runs immediately!");
})();
```

Output

```
This function runs immediately!
```

Explanation:
- The function is **defined**
- Then it is **executed immediately**

---

## Example with Variables

```javascript
(function () {
  let message = "Hello World";
  console.log(message);
})();
```

Output

```
Hello World
```

---

## Why We Use IIFE

- To **avoid polluting the global scope**
- To **execute code immediately**
- To create a **private scope**
- Often used in **module patterns**

---

## Structure of IIFE

```
(function(){
    // code
})();
```

or

```
(() => {
   // code
})();
```

---

## Short Interview Answer

An **Immediately Invoked Function Expression (IIFE)** is a function in JavaScript that **executes immediately after it is defined**, usually used to create a private scope and avoid global variable pollution.
