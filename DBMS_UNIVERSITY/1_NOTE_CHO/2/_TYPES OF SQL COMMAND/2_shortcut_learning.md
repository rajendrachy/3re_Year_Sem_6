## SQL Commands Learning Note

*   **DDL** -> **C**, **A**, **R**, **T**, **D** (Create, Alter, Rename, Truncate, Drop)
*   **DQL** -> **S** (Select)
*   **DML** -> **I**, **U**, **D** (Insert, Update, Delete)
*   **DCL** -> **G**, **R** (Grant, Revoke)
*   **TCL** -> **ST**, **C**, **R** (Start Transaction, Commit, Rollback)
