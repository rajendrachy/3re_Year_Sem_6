## 16. What is the `useReducer` Hook and when is it preferred over `useState`?

### What is `useReducer`?

`useReducer` is a React Hook used for **managing complex state logic** in functional components. It works similar to the **reducer function in Redux**. Instead of updating state directly, you dispatch actions that describe what should change, and a reducer function updates the state accordingly.

### Syntax

```javascript
const [state, dispatch] = useReducer(reducer, initialState);
```

* **state** – the current state value.
* **dispatch** – function used to send actions.
* **reducer** – a function that determines how the state changes.
* **initialState** – the starting value of the state.

### Example

```javascript
import React, { useReducer } from 'react';

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

function Counter() {
  const [state, dispatch] = useReducer(reducer, { count: 0 });

  return (
    <div>
      <p>Count: {state.count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>+</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>-</button>
    </div>
  );
}

export default Counter;
```

### When is `useReducer` Preferred Over `useState`?

Use **`useReducer`** when:

* The state logic is **complex**.
* Multiple state values depend on each other.
* State updates involve **multiple actions**.
* You want **better organization of state logic**.

### Key Differences

| Feature        | useState            | useReducer          |
| -------------- | ------------------- | ------------------- |
| Best for       | Simple state        | Complex state logic |
| Update method  | Direct state update | Dispatch actions    |
| Structure      | Simple              | More structured     |
| Logic handling | Inside component    |                     |
